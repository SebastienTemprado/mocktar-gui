# mocktar-gui
GUI for [Mocktar](https://github.com/SebastienTemprado/mocktar)

This project uses :

- Vue 3
- EsLint + Prettier
- Jest
- Cypress

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files

